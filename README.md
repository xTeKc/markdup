# markdup
A markdown to html compiler.

<br>

__Usage__ :
- `clone repo`
- `cargo build`
- place *.md* file inside dir
- `cargo run`
- outputs equivalent *.html* file

<br>

__Capabilities__ :
- converts `#` to `<h1></h1>` tags
- converts `plain text` into `<p></p>` tags

<br>
